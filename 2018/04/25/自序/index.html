[流程]{ :medium } { #heading }

  • hexo init提示无hexo运行环境

    : npm install并不能解决问题，网上也查了很多，原来是node版本过低，各种教程试了十多遍确实很绝望

  • hexo deploy失败，与git连接拒绝
    : 配置git本机与远程的ssh秘钥，可免除部署时的用户登录
    : repository设置为 `git@github.com:flyfish1024/flyfish1024.github.io.git`

  • hexo init所在目录不为空
    : init要在空目录，之后再npm install即可
