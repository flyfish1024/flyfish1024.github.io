[流程]{ :medium } { #heading }

  • hexo init提示无hexo运行环境

    : npm install并不能解决问题，网上也查了很多，原来是node版本过低，各种教程试了十多遍确实很绝望

  • hexo deploy失败，与git连接拒绝
    : 配置git本机与远程的ssh秘钥，可免除部署时的用户登录
    : repository设置为 `git@github.com:flyfish1024/flyfish1024.github.io.git`

  • hexo init所在目录不为空
    : init要在空目录，之后再npm install即可

原因

  • 导火索
    : 同事：诶，你们没事可以看看我的博客[(快看我有博客啦~NB不)]{ :underline }

    : 再其次，我也可以，诶，你们没事可以看看我的Blog~

  • 重要性
    : 我喜欢将一些东西记录下来，不是用神经元的那种~

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: true
    tags: true
